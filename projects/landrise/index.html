  • Extract File 7z yang ada di folder Project

  • Klik Add
  • Arahkan Ke Folder Hasil Extract Tadi
  • Setelah Project Berhasil terbuka,
  • Buka Folder Scene
  • Klik 2x salah satu scene yang tersedia

Video Cara Membuka Project

Pastikan menngunakan versi Unity sesuai project, agar tidak terjadi error saat mencoba membuka project
dan pastikan terhubung dengan koneksi internet untuk mendownload package yang di perlukan secara otomatis

*Hanya bisa di buka via Unity Hub, jika di copy paste sendiri, maka akan terjadi error karena dependecies yang di perlukan tidak dapat terinstall

Penjelasan beberapa Folder di dalam Project:

  • Data:

    Berisi File Mentah (Raw) yang akan di gunakan pada project

  • Prefab:

    Berisi File Pre-Set yg sudah di konfigurasi

  • Scenes:

    Berisi Scene/level yang tersedia

  • Scripts:

    Berisi Script yg digunakan untuk project utama

  • KenTank:

    Berisi Asset atau Tools pre-build yang telah saya kembangkan secara pribadi

    boleh di rename dengan nama sendiri (jika perlu)

  • External Assets: (Jika Ada)

    Berisi External Asset dari luar seperti GitHub, Asset Store, dan lainnya.
    bisa digunakan sebagai sumber atau referensi untuk laporan
    Jangan di perjual belikan untuk di bagian ini, krn lisensi di lindungi hukum Internasional

  • FMOD: (Jika Ada)

    Berisi Project Third-Party untuk Audio Studio yg terhubung secara langsung dengan Unity,
    tujuan nya agar suara yg di hasilkan lebih baik dari pada Audio bawaan Unity nya
    memerlukan pemahaman teknis tentang audio softwaring untuk edit file ini

Kusus untuk folder yang bernama "Resources",
jangan pernah sentuh folder ini, folder ini berisi file yang semuanya saling terhubung satu sama lain menggunakan path,
jika ada satu file saja yang hilang atau berpindah, game atau bahkan Unitynya akan error

Untuk cara Update Project:

  • Download file Update
  • Buka Unity Project sebelumnya
  • Klik Assets > Import Package > Custom Package > File Update (contoh Update 1.1.unitypackage)
  • dan Klik Import
untuk update, pastika di lakukan secara urut versi
contohnya, sebelum update versi 1.2, harus sudah update dulu ke 1.1 projectnya,
jika tidak di lakukan secara urut, nanti file nya akan tidak sinkron antara file baru ama file lama yang menyebabkan error
